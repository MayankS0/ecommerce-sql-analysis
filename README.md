# # 🛒 E-commerce SQL Analysis Project

## 📌 Overview
This project analyzes an e-commerce dataset using SQL to extract business insights such as customer behavior, revenue trends, retention, and segmentation.

---

## 🎯 Objectives
- Analyze key business KPIs
- Understand customer behavior
- Perform cohort and retention analysis
- Segment customers using RFM analysis

---

## 🛠 Tools Used
- SQL (Joins, CTEs, Window Functions, Aggregations)

---

## 📊 Key Analysis Performed

### 🔹 Business KPIs
- Total Revenue
- Total Orders
- Average Order Value

### 🔹 Customer Analysis
- New vs Returning Customers
- Customer Lifetime Value (CLV)
- Top Customers

### 🔹 Product Analysis
- Revenue by Category
- Top Selling Products

### 🔹 Time Analysis
- Monthly Revenue
- Running Revenue

### 🔹 Retention Analysis
- % Repeat Customers

### 🔹 Cohort Analysis
- User retention over time

### 🔹 RFM Segmentation
- Customer segmentation based on behavior
